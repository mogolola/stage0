Components of web:

The major pieces who make the world wide web work are computers, the internet and the servers. First, the browser working on a computer will send a request to the servers through the internet, using a protocol named http. Then there will be a feedback from the servers to the computer far-end and the browser will display certain contents as it should.

Components of web:

Hypertext markup language, it's the main type of document on the web .html can combine all kinds of media like text, image, video all together and format them into a more proper form to display in the web. This video gives a good overview.

Tags:

Tags are very important when wrritting html. Each tag has a function which help to display visual elements or text. How to use a tag? Normally it has a opening tag and a closing tag, contents are between them. The whole pieces(including the opening tag , contents and the closing tag) are called elements.

Computers are stupid.

Unlike the human brain the computer can hardly deal with the ambiguity with the context on time. They interpret the language literally. So we have to be careful not to make mistake which probably would cause a huge problem.

What is an inline element and a block element?

Html elements are divided into inline elements and block elements. The main difference between them is that the inline element doesn't change the form of the content and the block element will generate an invisible block. We will be able to define the style of the contents inside the box.
